Calcul des Indemnités Journalières (IJSS)

Un outil interactif pour maîtriser l'indemnisation des arrêts de travail.

Règles à jour au 1er juillet 2025.

Cette section détaille les règles d'indemnisation pour un arrêt maladie d'origine non professionnelle. Utilisez le simulateur pour estimer les IJSS en fonction du salaire brut et de la durée de l'arrêt, en tenant compte du délai de carence et du nouveau plafond de 1,4 fois le SMIC.

Délai de Carence

3 jours calendaires. L'indemnisation par la CPAM débute au 4ème jour d'arrêt. (Art. L. 323-1 Code de la Sécurité Sociale)

Taux d'Indemnisation

50% du Salaire Journalier de Référence (SJR). (Art. L. 323-4 Code de la Sécurité Sociale)

Plafonnement

Le salaire brut est limité à 1,4 fois le SMIC mensuel pour le calcul du SJR. (Art. L. 323-4 Code de la Sécurité Sociale)

L'indemnisation pour un Accident du Travail (AT), un Accident de Trajet ou une Maladie Professionnelle (MP) est plus favorable. Il n'y a pas de délai de carence, et les taux d'indemnisation sont progressifs. Notez que le jour de l'accident est intégralement payé par l'employeur.

Délai de Carence

0 jour. L'indemnisation débute dès le lendemain de l'accident. (Art. L. 433-1 Code de la Sécurité Sociale)

Taux d'Indemnisation

60% du SJR les 28 premiers jours.

80% du SJR à partir du 29ème jour. (Art. R. 433-5 Code de la Sécurité Sociale)

Base de Calcul

Le SJR est calculé sur le salaire brut du mois précédant l'arrêt, divisé par 30,42. (Art. R. 433-5 Code de la Sécurité Sociale)

Les congés liés à l'arrivée d'un enfant (Maternité, Paternité, Adoption) bénéficient d'une indemnisation intégrale du salaire journalier de référence, sans délai de carence. Le calcul du SJR est spécifique pour se rapprocher du salaire net.

Délai de Carence

0 jour. L'indemnisation débute le premier jour du congé. (Art. L. 331-3 Code de la Sécurité Sociale)

Taux d'Indemnisation

100% du Salaire Journalier de Référence. (Art. L. 331-3 Code de la Sécurité Sociale)

Base de Calcul

Le SJR est calculé sur la moyenne des 3 derniers salaires bruts (limités au PMSS), après un abattement de 21%. (Art. R. 331-5 Code de la Sécurité Sociale)

Suite à la loi du 22 avril 2024, un salarié en arrêt maladie non professionnelle acquiert désormais des congés payés. Cette section vous explique ces nouvelles règles et vous permet de calculer les jours acquis.

Taux d'Acquisition

2 jours ouvrables par mois d'arrêt maladie non pro. (Art. L. 3141-5 Code du travail)

Plafonnement Annuel

L'acquisition est limitée à 24 jours ouvrables par an au titre de la maladie. (Art. L. 3141-5 Code du travail)

Report des Congés

Le salarié dispose de 15 mois après son retour pour prendre les congés non pris, suite à l'information de l'employeur. (Art. L. 3141-28-1 Code du travail)
